# dev‑marketer – Real‑Time Launch Analytics (demo)

A tiny, self‑contained Python library that aggregates click, email‑open,
signup and paid‑conversion events per traffic source.  
It can:

* Record events in‑memory.
* Produce a snapshot with totals and per‑source breakdowns.
* Compare the snapshot against an external API payload with a configurable
  tolerance (default ±5%).
* Tell whether a dashboard should refresh (default every 30 seconds).
